# Polluter
A menu-bar app that replaces the newline characters in .txt files with `'\r\n'`. This allows them to work with Windows Notepad.

Why is this important? It's not, but it's a good practice program.

# Usage
Basic instructions.

## Install packages
**Yarn**
```
yarn
```

**NPM**
```
npm install
```
